Pour utiliser le Rummikub : 
	-Se déplacer dans le dossier
	-Lancer la commande "./commande.sh"
	-Lancer l'executable "Rummikub"

Pour la doc, il faut lancer le fichier index.html du dossier "Doc"
	
Lors de nos tests sur Qt creator, le programme fonctionnait. En revanche, une fois passé sur les machines de la fac, la compilation ne marchait plus bien que nous gérons les dépendances dans notre bash.
